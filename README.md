## 📋 What is kAIhoot?

kAIhoot is a Chrome extension designed to help you answer Kahoot questions automatically. It works with all question types, including multiple choice, true/false, and puzzles.

The extension uses artificial intelligence to understand questions and find the best answers fast. It runs inside your browser and interacts with Kahoot games directly.

kAIhoot can improve your quiz experience by saving time and effort during fast-paced games. It works on Windows computers with Google Chrome installed.

---

## 🖥️ System Requirements

- Windows 10 or later  
- Google Chrome browser (latest version recommended)  
- Stable internet connection  
- At least 100 MB of free disk space  

Chrome extensions run inside the browser, so you don’t need to install anything outside of Chrome.

## 🚀 Getting Started

Follow these steps carefully to get kAIhoot up and running on your Windows PC.

The page contains the latest files and installation instructions.

### 2. Download the Extension Files

On the GitHub page, look for the section named **Releases** or **Assets**. This is where the extension files are kept.

You need to download the `.zip` or `.crx` file for the Chrome extension. Usually, the `.zip` file contains the unpacked extension files.

Save the file in an easy-to-find folder on your PC, such as your **Downloads** folder.

### 3. Extract the Files

If you downloaded a `.zip` file, right-click it and choose **Extract All**.

Pick a folder location that is easy to remember. You will need this folder in the next step.

### 4. Install the Extension in Chrome

Open Google Chrome.

In the address bar, type:

```
chrome://extensions/
```

and press Enter.

In the top right, turn on **Developer mode** by clicking the switch.

Click the **Load unpacked** button.

Navigate to the folder where you extracted the files. Select it and click **Select Folder**.

The kAIhoot extension will now appear in your list of Chrome extensions.

Make sure it is enabled.

### 5. Authorize Permissions

The extension may request permission to read data on Kahoot quiz pages. Click **Allow** so it can function correctly.

## 🧩 How to Use kAIhoot

1. Open the Kahoot game page in Chrome.

2. Start or join a Kahoot quiz.

3. When questions appear, kAIhoot will analyze them.

4. The extension will highlight or select the answer it finds best.

5. You can choose to follow the suggested answers or override manually.

The whole process happens quickly. You do not need to switch tabs or do anything extra once you start.

## ⚙️ Custom Settings

The extension comes with settings you can adjust:

- **Auto-Answer Mode:** Choose between fully automatic or manual confirmation before answering.  
- **Answer Speed:** Set how fast answers should be selected.  
- **Question Types:** Enable or disable support for specific question formats like puzzles or true/false.  
- **Notification:** Turn on/off alerts when answers are selected.  

Access settings by clicking the kAIhoot icon next to Chrome’s address bar.

---

## 🔍 Troubleshooting Tips

- Make sure your Chrome browser is up to date.
- Check that the kAIhoot extension is enabled in Chrome extensions manager.
- Refresh the Kahoot game page after installing the extension.
- Restart Chrome if the extension doesn’t respond.
- Disable other Chrome extensions that might conflict with kAIhoot.
- Confirm that your internet connection is stable.
- Reinstall the extension if you see errors.

kAIhoot works locally in your browser. It does not send your quiz data to external servers. Your personal information and game data remain on your PC.
